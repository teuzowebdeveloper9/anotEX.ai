# anotEX.ai

🎓 Aplicação inteligente para anotações de aulas com captura de áudio em tempo real e transcrição automática.

```
anotEx.ai/
├── ai-docs/          # Documentação de pesquisa e decisões técnicas
```

## 🚀 Funcionalidades Planejadas

- 🎤 Captura de áudio em tempo real (microfone)
- 🖥️ Captura de áudio do sistema (aulas online)
- 📝 Transcrição automática com IA
- 📚 Organização de anotações por aula/disciplina
